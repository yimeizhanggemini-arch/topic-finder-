# 选题猎手 · topic-finder

把一个关键词 + 你自己的想法,加工成可以直接开做的「选题包」——面向小红书 / 抖音创作者。

## 这是什么

市面上"找热门选题"的工具已经很多,热榜谁都能抓。这个工作流解决的是更难、也更值钱的下一步:**把一个通用热点,翻译成"适合你这个账号、这个平台、这个目标,而且别人还没做透"的选题。**

它做四件通用工具不做的事:

- **先问你**:开场摸清你手上有什么(光一个词,还是你也有观点 / 亲身经历)——因为"真实体验""独特视角"这些只能来自你。
- **分干货 / 情绪两条道**:两类内容的判断标准完全不同,分开核查。
- **挖评论区**:从爆款评论里的"提问"和"分歧"挖真实选题,而不只是抄热榜。
- **逼你确认**:最后用一份清单逼你过一遍"这选题到底属不属于你"。

## 文件结构

| 文件           | 作用                                              |
| ------------ | ----------------------------------------------- |
| `SKILL.md`   | 核心方法论(六步流程)。AI 执行的就是这个。                         |
| `profile.md` | 你的创作者档案(平台 / 赛道 / 目标 / 人设 / 选题历史)。**换号只改这个文件。** |
| `AGENTS.md`  | 给 Codex 等编程 agent 的入口:何时触发 + 指向 SKILL.md。       |
| `README.md`  | 你正在看的这个。                                        |

## 安装(先把文件下载到本地)

在 Codex 里用之前,得先把这个仓库弄到你自己电脑上。两种方式,挑一种:

**方式一:下载 ZIP(不会 git 也能用,推荐给非技术的人)**

在本仓库页面点右上方绿色的 **Code** 按钮 → **Download ZIP** → 解压到任意位置,会得到一个 `topic-finder-` 文件夹。

**方式二:git clone(熟命令行的话更快)**

```bash
git clone https://github.com/yimeizhanggemini-arch/topic-finder-.git
```

两种方式下载完,都会得到一个 `topic-finder-` 文件夹,里面就是上面「文件结构」列的那几个文件。接着照下面「在 Codex 里怎么用」操作即可。

> 不熟终端的话,有个进文件夹的笨办法:打开终端,输入 `cd ` 加一个空格,再把 `topic-finder-` 文件夹直接拖进终端窗口,回车,就进到这个目录了。

## 依赖

这个工作流需要一个能返回小红书 / 抖音数据的工具。两种用法,二选一:

**A. 装数据工具(自动取数,体验最好)**

推荐 **Agent Reach** —— 开源、免费、支持 Codex:

1. 安装:`pip install agent-reach`,再运行 `agent-reach install`(仓库:https://github.com/Panniantong/Agent-Reach)。
2. 小红书需要登录态:按它的引导从浏览器导出 cookie。**建议用小号,别用主账号。**
3. 装完用 `agent-reach doctor` 看哪些渠道已就绪。

> Agent Reach 当前主要解决小红书;抖音暂时没有稳定的自动取数后端,会走下面的「手动模式」。

**B. 不装工具,用手动模式**

如果你不想配 cookie、或只是想先试试,可以不装任何工具。这时工作流会让你自己去平台搜一下,把前 5–8 条爆款的标题 + 赞 / 藏 / 评 / 转、再加几条高赞评论贴给它,它照样能跑完整套分析。数据工具的作用,只是把"你手动复制这一步"自动化了而已。

## 在 Codex 里怎么用

1. 把本文件夹放进你的 Codex 项目(装了数据工具自动取数体验最好;不装也能用手动模式,见上面「依赖」)。
2. 在这个文件夹里启动 Codex——它会在 session 开始时**自动读取 `AGENTS.md`**,无需手动加载。
3. 直接用大白话提需求,例如:「帮我找选题,关键词是 XXX」。Codex 会按 `SKILL.md` 的流程走:先和你开场对话,再一步步产出选题包。

> 想确认指令加载没?在 session 里问一句「总结下你这个 session 加载了哪些指令」即可。

## 怎么定制

打开 `profile.md`,把带 ⚠️ 的几栏填成你自己的。最值钱的是「人设」和「选题历史」两栏——前者决定差异化角度,后者让它越用越懂你。主流程 `SKILL.md` 不用动。
